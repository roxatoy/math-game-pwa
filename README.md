# 🧮 МатИгра - PWA

Интерактивная математическая игра как прогрессивное веб-приложение (PWA). Работает онлайн и оффлайн на всех устройствах! 📱💻

## ✨ Возможности

### 🎮 Игровые режимы
- **Тренировка** - 5 вопросов (начните отсюда)
- **Нормально** - 10 вопросов (классический режим)
- **Челлендж** - 20 вопросов (для опытных)
- **Экстрим** - 30 вопросов (максимум сложности)

### 📚 Математические операции
- ➕ Сложение
- ➖ Вычитание
- ✖️ Умножение
- ➗ Деление

### 🏔️ Уровни сложности
- 🌱 **Лёгкий** - числа до 10
- 🌟 **Средний** - числа до 20
- 🔥 **Трудный** - числа до 50
- 💥 **Безумный** - числа до 100

### 🏆 Система достижений
- 👣 **Первый шаг** - заработайте 5 очков
- 👏 **Идеально!** - 100% точность
- ⚡ **Молния** - ответьте за 1 сек
- 🌊 **На волне** - стрик из 10 ответов
- 💰 **Миллионер** - 1000 очков всего
- 🏆 **Мастер** - сыграйте 50 игр

### 📊 Статистика и сохранение
- Всего очков
- Количество сыгранных игр
- Максимальный стрик
- Самый быстрый ответ
- Данные сохраняются локально (localStorage)

### ⚙️ Настройки
- 🔊 Управление звуком
- 📊 Просмотр подробной статистики
- 🗑️ Сброс всех данных

### 🔊 Звуковые эффекты
- ✅ Приятный звук для правильных ответов
- ❌ Тревожный звук для ошибок
- 🔕 Управление звуком в настройках

## 🚀 Быстрый старт

### Локально (Python)
```bash
python -m http.server 8000
# Откройте http://localhost:8000 в браузере
```

### Локально (Node.js)
```bash
npm install -g http-server
http-server
# Откройте http://localhost:8080 в браузере
```

### Развертывание на GitHub Pages
1. Создайте репозиторий `username.github.io`
2. Скопируйте все файлы туда
3. Приложение будет доступно по ссылке: `https://username.github.io`

## 📱 Установка как приложение

### На Android
1. Откройте приложение в Chrome
2. Нажмите кнопку меню (⋮)
3. Выберите "Установить приложение"
4. Согласитесь с установкой

### На iPhone/iPad
1. Откройте приложение в Safari
2. Нажмите кнопку "Поделиться" (↗)
3. Выберите "На главный экран"
4. Введите имя приложения

## 🛠️ Технологический стек

- **React 18** - UI фреймворк
- **Babel** - транспиляция JSX
- **Service Worker** - оффлайн функциональность
- **localStorage** - сохранение данных
- **Web Audio API** - звуковые эффекты

## 📈 Как улучшить результаты

1. **Выбирайте нужные операции** - сосредоточьтесь на одной сначала
2. **Начните с лёгкого уровня** - постепенно повышайте сложность
3. **Играйте регулярно** - это поможет улучшить навыки
4. **Соревнуйтесь с собой** - старайтесь превзойти свои результаты
5. **Разблокируйте достижения** - это дополнительная мотивация

## 🎁 Система бонусов

- **Базовый бонус**: 1 очко за правильный ответ
- **Бонус стрика**: +1 очко за ответ, если стрик >= 2
- **Бонус времени**: +1 очко если ответили за 8+ сек
